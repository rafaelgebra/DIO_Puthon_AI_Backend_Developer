Introdução

Python fornece várias maneiras de ler um arquivo. Podemos usar 'read()', 'readline()' ou 'readlines()', dependendo de nossas necessidades. Exemplos "read" file = open('example.txt', 'r') print(file.read()) file.close()

Métodos readline e readlines
